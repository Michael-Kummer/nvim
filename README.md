# nvim

Set up to be modular where each plugin is in their own file.  
Almost a barebones setup. 


## Directory Structure

```text
nvim
├── README.md
├── init.lua
├── lazy-lock.json
└── lua
    ├── config
    │   └── lazy.lua
    └── plugins
        ├── devicons.lua
        ├── example.lua
        ├── gitsigns.lua
        ├── mason.lua
        ├── minis.lua
        ├── quicknotes.lua
        ├── riscv.lua
        ├── theme.lua
        └── todo-comments.lua
```
